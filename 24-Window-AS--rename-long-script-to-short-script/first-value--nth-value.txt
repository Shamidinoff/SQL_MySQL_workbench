1)) Для чего предназначены и чем между собой отличаются функции first_value и NTH_value?

Предназначение одинаковое - взять значение из определенной ячейки\строки в столбце.

Но First_Value в качестве аргумента требует только столбец, поскольку всегда берет первое значение (из первой строки) по указанному столбцу в заданном окне.

А NTH_Value  кроме столбца требует еще один аргумент, а именно номер строки из которой в этом столбце нужно взять значение. 

2)) Для чего предназначены и чем между собой отличаются функции LEAD и LAG?

Предназначение одинаковое - взять значения из определенных ячеек\строк. Обе требуют 2 обязательных аргумента: столбец и номер строки\ячейки в нем. 

Причем если строки из которых нужно взять значение находятся за пределами окон, то подставляет null. И этот null  при необходимости можно заменить любым другим значением, внеся в скобки функции третий аргумент (значение, которым нужно заменить null).

Но важное отличие в том, что LEAD берет значение из последующих\будущих строк. А LAG берет значения из предшествующих\прошлых строк.

4)) А можете сказать, что будет если использовать данные оконные функции смещения без over()?

Ничего не будет, точнее ошибка будет - поскольку данные функции существуют для БД только как оконное вычисление вместе с over().
